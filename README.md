## Installation and Setup Instructions

#### Local Setup

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Run Test Suite:

`npm test`

To Start Server:

`npm start`

To Visit App:

`localhost:3000`

#### Services

Links to the deployed applications:
